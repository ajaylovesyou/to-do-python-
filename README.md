# To Do List Project in Python

This is a simple to do list project in Python. We will create a command line application that allows users to add, view, and delete tasks from their to do list.

## Features

- Add a task to the to do list
- View all tasks in the to do list
- Delete a task from the to do list

## Usage

1. Clone the repository to your local machine.
2. Run the `todotask.py` file to start the application.
3. Follow the prompts to add, view, or delete tasks from your to do list.

## Future Improvements

- Add a feature to mark tasks as completed.
- Implement a user interface for better user experience.
- Save tasks to a file for persistence.
